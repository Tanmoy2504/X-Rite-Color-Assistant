# X-Rite Color Assistant

X-Rite Color Assistant Application for Lenovo **ThinkPad** / **Legion** series laptops with 100% sRGB displays.  
Your Lenovo laptop model may or may not be supported.  
Tested with **Legion 5 Pro (16ARH7H)**.
