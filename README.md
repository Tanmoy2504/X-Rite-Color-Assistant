# X-Rite-Color-Assistant
X-Rite Color Assistant Application for Lenovo Thinkpad / Legion series of laptop which has 100% sRGB Display. Your Lenovo laptop model may or may not be supported... Tested with Legion 5 Pro (16ARH7H). 

This application is same across the Thinkpad and Legion Lineup so install and check for updates (If Any). During the first Install , go to settings inside the app and press "Restore Profile". make sure the app is running as administrator or
else this will result in failure / error. The administrator permission is required to download the ICC color Profiles archive (sRGB, HDR, REC 709 etc.) and install, convert the profiles into system directory.

This windows Application was extracted directly from the update file.

This file is signed by X-Rite Incorporated and is in no way modified or cracked. 

The reason this file is provided because many models of Legion / Thinkpad doesn't have this application under drivers and download secion inside official application and it is especailly a problem when the computer needs to be
formatted / clean installation is needed...  The availablity of this file is so that you don't have to deal with Lenovo Tech Support and spend useless hours trying to download from pcsupport.lenovo.com by entering serial number just to download this file

The Software version is version 2.64.0
